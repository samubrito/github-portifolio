Projetos

IBGE Unemployment Pipeline: ETL & Predictive Modeling

ETL | Machine Learning | Github Actions | Power BI | Requests

Esta solução automatiza a coleta de dados públicos de desemprego e supre a ausência de modelos para antecipação de tendências de mercado. A arquitetura consiste em um pipeline ponta a ponta que consome a API do IBGE via Python, aplica engenharia de atributos e treina um modelo de Regressão Linear. Com orquestração via GitHub Actions (CI/CD), o sistema entrega monitoramento preditivo semanal e métricas de erro (MAE) em um dashboard dinâmico, garantindo total autonomia e precisão na análise de dados econômicos.

Projeto Power BI

Supply Chain Analytics

Power BI | Linguagem M | Linguagem Dax | KPIs | Storytelling

O projeto resolve a fragmentação de dados logísticos e financeiros dispersos em múltiplas fontes, que dificultavam a análise ágil de performance. Foi implementado um pipeline de ETL escalável em Linguagem M com funções personalizadas para limpeza dinâmica de dados, alimentando dashboards estratégicos no Power BI. O resultado é a centralização de KPIs como Lead Time, lucratividade e riscos de fraude, permitindo a identificação imediata de gargalos operacionais e suporte à decisão estratégica em tempo real.

Projeto Power BI

Automação de Consolidação: Relatório Operacional Logístico

Python | Pandas | ETL | Automação

Este projeto aborda a inconsistência e fragmentação de dados provenientes de 25 arquivos Excel com mais de 100 abas de filiais mal estruturadas. Através de um algoritmo desenvolvido em Python (Pandas), o sistema realiza a varredura automatizada de diretórios, transposição de matrizes complexas e normalização de campos de custo e tempo de sistemas legados. A solução otimizou o processamento de várias horas de esforço manual para apenas alguns segundos, eliminando riscos de erro humano e garantindo a escalabilidade necessária para altas volumetrias de dados.
